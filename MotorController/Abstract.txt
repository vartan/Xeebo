The BLINKY project is a demo program using the NXP LPC11Uxx Microcontroller.

It demonstrates an BLINKY :

 This project tests two LEDs, two LEDs are blink one after another.

 

The BLINKY program is available for the following targets:
  NXP LPC11Uxx Eval Board: configured for on-chip Flash ROM
                 (used for production or target debugging)
